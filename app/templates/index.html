rQES (Wallet Centric)

Welcome to the remote Qualified Eletronic Signature component wallet-centric backend server.

A Web GUI supporting the same-flow device for tests is available at: https://walletcentric.signer.eudiw.dev/tester

In this domain, it is made available the SCA Server and the QTSP Server composed by an Authorization Server and Resourse Server.

SCA Server Endpoints:

  • /credential/authorize
  • /signatures/signDoc
  • /credential/oauth/login/code

QTSP Server Endpoints:

  • Authorization Server
    • /oauth2/authorize
    • /oauth2/token
    • /oid4vp/callback
  • Resource server
    • /csc/v2/info
    • /csc/v2/signatures/signHash
    • /csc/v2/credentials/list
    • /csc/v2/credentials/info
